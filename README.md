📱 Project Title
Mobile Phone Feature Identifier
📖 About the Project
The Mobile Phone Feature Identifier is a simple software application that compares the specifications of two mobile phones (old and new models) and identifies the newly added or upgraded features.
This project helps users easily understand improvements in mobile technology without manually comparing specifications.
✨ Features
🔍 Compares old and new mobile phone features
⚡ Identifies newly added features
🔄 Detects upgraded specifications
🧾 Simple and easy-to-understand output
💻 Lightweight and beginner-friendly
⚙️ How It Works
The program stores features of an old mobile phone
Stores features of a new mobile phone
Compares both using a loop
Displays:
New features (not present before)
Upgraded features (changed values)
📊 Example Output

New Features in the Mobile Phone:

Display → Upgraded from LCD to AMOLED
Camera → Upgraded from 12MP to 50MP
Battery → Upgraded from 4000mAh to 5000mAh
Charging → Upgraded from Normal Charging to Fast Charging
Network → Upgraded from 4G to 5G
Fingerprint → New Feature Added
Face Unlock → New Feature Added
🛠️ Technologies Used
🐍 Python (Core Programming)
🧠 Basic Data Structures (Dictionary)
💻 Command Line Interface (CLI)
▶️ How to Run
Install Python (version 3.x)
Copy the code into a .py file (e.g., mobile_features.py)
Open terminal / command prompt
Run the program:

python mobile_features.py
